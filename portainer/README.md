# Portainer Docker
>Portainer是一个轻量级的管理UI，可让您轻松管理Docker主机或Swarm集群。

## 环境

- CentOS7
- Docker-ce 19.03.12
- Docker Compose 1.26.1
- Jenkins Docker Image portainer/portainer


## 启动Portainer容器

```
docker-compose up -d
```

或者

```
cd portainer/

sh run.sh
```
